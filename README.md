* mongoose
* jwt

### 项目运行
```
# 克隆
git clone https://github.com/qw572386/vit-blog.git

# 安装依赖

## 客户端依赖
cd client && npm install

## 服务端依赖
cd server && npm install

# 启动运行（前提：启动mongodb服务）

## 启动客户端
cd client && npm run dev

## 启动服务端
cd server && npm run dev

```
